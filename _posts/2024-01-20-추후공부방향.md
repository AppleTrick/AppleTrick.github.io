# 방향성

- Web 웹
- Moblie Web 모바일 웹
- Moblie App 모바일 앱 (하이브리드 앱), react-native 사용 가능 : 웹뷰를 사용
- Cross Flatform 크로스 플랫폼 : 안드로이드와 IOS 둘 다 동시에 개발 ex) react-native , flutter
- Android 안드로이드 ex) 코틀린
- IOS 애플 , ex) 스위프트 공부해야됨
